== Description ==

The Secure and Accessible Contact Form plugin for WordPress allows you to add a simple contact form to your website easily and hassle free. It comes with a few themes (you can also customise it with CSS), and can be inserted into any post or page via a shortcode.

= Features =

*	Easy to use form Overview page, with config information, and spam count.
*	Easily disable or enable address/phone fields
*	Add the form via a shortcode, a tag, or a PHP function.
*	Great security measures implemented, to prevent spam and high-jacking.
*	Form designed with accessibility and usability in mind.
*	Standards compliant code
*	Simple setup, with helpful prompts and extensive documentation.
*	Configuration built into the wordpress interface, no need to edit any code manually.
*	Easy to style, either using your own CSS files, our in-built editor, or a pre-made theme.
